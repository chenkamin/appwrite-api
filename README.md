# Appwrite-api

- Vehicles rest API 
- CRUD operations
- PHP - swoole - utopia



# To run the project

- clone the project
- run docker build -t appwrite-api .
- docker run -d -p 80:80 appwrite-api

# end points

- GET api/v1/vehicles
- GET api/v1/vehicles?id={ID_NUM}
- POST api/v1/vehicles
- PUT api/v1/vehicles?id={ID_NUM}
- DELETE api/v1/vehicles?id={ID_NUM}
